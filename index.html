EXPOSE SUR HTML

INTRODUCTION

HTML (HyperText Markup Language) : il a fait son apparition dès 1991 lors du
lancement du Web. Son rôle est de gérer et organiser le contenu. C'est donc en
HTML que vous écrirez ce qui doit être affiché sur la page : du texte, des liens, des
images… Vous direz par exemple : « Ceci est mon titre, ceci est mon menu, voici le
texte principal de la page, voici une image à afficher, etc. » ;

STRUCUTRES

Un document html est composé:
  1. D'une déclaration DOCTYPE HTML qui définit que le document est du HTML5
  2. Un élément HTML est un élément racine de la page
  3. Un élément HEAD contient des méta informations sur le document
  4. Un élément TITLE spécifie le titre du document
  5. Un élément BODY contient un contenu visible de la page

LES TITRES

Les titres sont définies avec les balises de h1 à h6. H1 définit le titre
le plus important et le H6 le moins important.

Exemple

H2

H3

H4

H5
H6
